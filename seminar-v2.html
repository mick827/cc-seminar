スキルの種類

Claude Code のスキルには2種類あります

GLOBAL SKILL
グローバルスキル
パソコン全体で使えるスキル。どのフォルダを開いていても呼び出せる。
💻 あなたのパソコン全体
~/.claude/skills/
📄 mail-reply.md
📄 sns-post.md
📄 summary.md
→📁 仕事フォルダで使える
→📁 個人フォルダでも使える
→📁 testフォルダでも使える
  • どのフォルダでも使える
  • パソコン全体に保存される
  • サイドバーには表示されない
  • 汎用的なスキルに向いている
PROJECT SKILL
プロジェクトスキル
そのフォルダの中だけで使えるスキル。フォルダを開いた時だけ使える。
📁 作業フォルダの中
📁 test/
.claude/commands/
📄 threads-post.md
📄 mail-reply.md
001_profile/
📄 about-me.txt
このフォルダを開いた時だけ使える
  • このフォルダの中だけで使える
  • .claude/commandsに保存される
  • サイドバーに表示される ✅
  • フォルダごとに管理しやすい
どちらを使う？
グローバルスキルが向いている場面
どの仕事でも繰り返し使うスキル
例：メール返信・議事録作成・要約など
フォルダを問わず使いたい時
プロジェクトスキルが向いている場面
特定のフォルダ専用のスキル
例：このフォルダのプロフィールを使ったThreads投稿
今日のハンズオンはこちら ✅
Claude Codeへの指示文
GLOBAL SKILL（グローバル）
今やった作業をスキル化して
グローバルスキルで保存して
どこからでも使いたい時
PROJECT SKILL（プロジェクト）
今やった作業をスキル化して
プロジェクトスキルで保存して
このフォルダ専用にしたい時 ✅
今日のハンズオンでは「プロジェクトスキル」を使います
